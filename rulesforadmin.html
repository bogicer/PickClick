правила для адміністрації проекту

🔹 1. Загальні обов’язки

  1. Адмін — це не бог, а наглядач за порядком.
  2. Головна мета — підтримувати стабільну роботу сервера, а не втручатися у гру.
  3. Заборонено використовувати адмінку для особистої вигоди або гри на користь друзів.
  4. Будь-які зміни в плагінах, світі чи налаштуваннях — тільки з дозволу головного адміністратора.

🔹 2. Поведінка

  1. Поводитися спокійно та нейтрально, навіть у конфліктах між гравцями.
  2. Заборонено погрожувати баном, образами чи приниженням.
  3. У спірних ситуаціях — збирати докази (скріни, логи), а не банити одразу.
  4. Якщо помилився — визнай це, головне — чесність.

     5. адміни не можуть заставляти гравців купляти донати, та                                                                           погрожувати баном


🔹 3. Використання команд

  1. /ban, /kick, /mute — лише за реальні порушення правил сервера.
  2. /give, /tp, /gamemode — заборонено використовувати для гри або допомоги гравцям.
  3. Режим Spectator дозволено лише для спостереження за порушниками.
  4. Жодних фейкових івентів чи “жартівливих” банів. (покарання

дуже жостоке)

🔹 4. Гриферство та PvP

  1. Адміни можуть грати як звичайні гравці, але без переваг адмінки.
  2. Якщо граєш — вимикай OP або адмін-режим.
  3. Заборонено грифити бази гравців, поки ти у ролі адміністратора.

🔹 5. Комунікація

  1. Усі дії адміністратора мають бути прозорими.
  2. Не приховуй порушення інших адмінів.
  3. Використовуй Discord/чат адмінів для узгодження дій.
  4. Повідомляй про будь-які збої, лаги, хакерів чи конфлікти.
  5. записуй весь робочий процес.

🔹 6. Покарання для адмінів

  1. За перше порушення — попередження.
  2. За повторне — зняття прав або тимчасовий бан.
  3. Зловживання OP — перманентне видалення з команди.

🔹 7. Принципи

  • 🧱 Справедливість понад усе.
  • ⚔️ Гриферство дозволено, але не з адмінки.
  • 💬 Поважай гравців, навіть якщо вони твої вороги.
  • 🔒 Адмінка — це відповідальність, а не сила.
